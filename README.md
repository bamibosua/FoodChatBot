# How To Run

## 1. Install dependencies from `requirements.txt`

Please install all required libraries using the following command:

``` bash
pip install -r requirements.txt
```

## 2. Open the Notebook

Open the notebook at the following link:

https://colab.research.google.com/drive/1wSFFxLNc8T7uZSnSQ0Zgnsl2-_whuGCm

Run the notebook and follow the instructions in Google Colab.
After the deployment link is generated, copy the link and paste it into the `client.py` file located in `MLPModule/ai`.

## 3. Run `main.py`

After installing the dependencies, run:

```bash
python3 -m streamlit run main.py
```
